# 1. branch

```bash
git branch // 어떤 branch가 있는지, 어떤 branch에 있는지 확인
git branch branch1 // branch1이라는 이름의 branch를 생성
git branch -d branch1 // branch1이라는 이름의 branch를 삭제
git checkout branch1 // branch1으로 이동
git checkout -b branch1 // branch1이라는 이름의 branch를 생성하면서 동시에 branch1으로 이동
git push origin branch1 // github에 branch1을 push
```
